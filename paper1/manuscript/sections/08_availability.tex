\section*{Data Availability}
All datasets used in this study (CIFAR-10, CIFAR-100, and Flowers-102) are publicly available through standard PyTorch torchvision and HuggingFace dataset interfaces. The source data underlying the main-text and Supplementary figures and tables are packaged with the submission bundle together with the corresponding manifests, logs, and figure-generation inputs. A submission-matched reproducibility archive containing the code snapshot, source data, SHA-256 manifest, and \texttt{CITATION.cff} file accompanies the reviewer materials; the same archive is staged for Zenodo deposition, with the DOI field reserved for acceptance-stage registration.

\section*{Code Availability}
The reviewer-accessible archive contains the simulation framework, training and evaluation scripts, device-profile utilities, plotting code, and submission-matched source data needed to reproduce the figures and tables. The codebase is distributed under the MIT licence, and the curated public repository is staged at \url{https://github.com/Leslie360/HAT}; Zenodo registration is reserved for the acceptance-stage public release.
