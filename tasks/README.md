# Pipeline Task Files

These Markdown files are pipeline execution specs and historical dispatch records.

They are not the canonical source for current numerical truth. Some older task specs intentionally preserve the wording and numbers that were known at the time of dispatch, including values later superseded by audit.

Canonical references after 2026-04-30:

- `report_md/_gpt/CLAUDE_OPUS_FINAL_DIRECTION_BRIEF_20260430.md`
- `scripts/_gpt/check_locked_numbers.py`
- `report_md/_gpt/AGENT_SYNC_gpt.md`

Before using a number from any task file in manuscript text, verify it against the locked-number guard or the relevant JSON/source summary.
