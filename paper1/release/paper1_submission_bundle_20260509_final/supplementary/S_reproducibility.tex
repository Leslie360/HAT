\section*{Supplementary Note S-Reproducibility}
\label{supp:reproducibility}

The reviewer-accessible reproducibility archive contains the source code for the simulation framework, the generated device profiles, the core analog-CIM operator implementations, the hardware-aware training (HAT) scripts, and the evaluation pipelines for the physical non-idealities studied here. These files are bundled with submission-matched source data and SHA-256 manifests; the curated public repository and Zenodo DOI will be finalized at acceptance.
