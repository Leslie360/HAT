## 3. Key Findings

1. **Retention improves last1 performance.** Counter-intuitively, applying retention decay (rst > 0) reduces PPL by ~0.3 for last1. This is likely because retention attenuates older, noisier KV values, acting as a temporal regularizer.
2. **Retention saturation.** All rst values >0 converge to the same PPL because the double-exponential decay `A_1*exp(-age/tau_1) + A_2*exp(-age/tau_2) + A_0` quickly saturates at the persistent fraction `A_0 = 0.6`. Once age exceeds a few tau, further increases in rst have negligible effect.
3. **All-layers is catastrophic with retention.** When all 24 layers suffer retention decay, PPL explodes from 35.8 to ~177. This confirms retention stress is a severe failure mode for wide analog KV deployment, but manageable with selective last1.
4. **Digital baseline note.** Digital baseline PPL = 22.18, higher than the 15.68 figure cited in prior documents. This discrepancy warrants investigation but does not affect the relative comparison between analog routes.
