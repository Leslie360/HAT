\section*{Supplementary Note S-Reproducibility}
\label{supp:reproducibility}

To facilitate the reproduction and extension of the results presented in this study, the full source code for the simulation framework and the generated device profiles will be made available. This includes the core analog-CIM operator implementations, the hardware-aware training (HAT) scripts, and the evaluation pipelines for various physical non-idealities. All materials are hosted at an anonymous repository to be disclosed upon acceptance.
