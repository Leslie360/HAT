# 2. 相关工作

## 2.1 有机突触器件的研究现状
在过去的十年中，有机光电突触器件和有机电化学晶体管（OECT）在神经形态计算领域表现出了巨大潜力。其突出的低电压运行、可调谐的模拟电导态以及对光刺激的仿生感知识别能力，使其成为边缘人工智能（Edge AI）的理想硬件。然而，现有的材料研究多集中于单一器件的表征或简单的逻辑门模拟。例如，尽管有研究报道了基于 TIPS-Pentacene 的超高动态范围和多态存储能力，但其对复杂视觉识别任务（如 ImageNet 或 Vision Transformer 分类）的鲁棒性尚未得到充分验证。

## 2.2 存算一体（CIM）仿真工具
目前的 CIM 仿真工具，如 DNN+NeuroSim 和 MemTorch，已经为无机阻变存储器（RRAM）和相变存储器（PCM）建立了较为成熟的建模体系。然而，这些工具往往缺乏对有机半导体器件特有的物理非理想性（如双指数保持力衰减、光电流亚线性响应以及特定的非线性写入动力学）的针对性支持。尤其是在处理混合模拟-数字架构下的异构视觉模型（如混合卷积-Transformer 架构）时，现有的仿真框架难以直接建立从物理参数到系统能效的精确链路。

## 2.3 硬件感知训练（HAT）
硬件感知训练（Hardware-Aware Training）通过在训练过程中将量化、噪声和失配等物理约束显式地包含在模型的前向传播路径中，从而诱导模型学习对硬件非理想性具有容忍度的权重分布。在无机存算一体领域，HAT 已被证明能显著提升推理精度。然而，在有机光电存算一体系统上应用 HAT，并针对 Transformer 这种对权重分布极其敏感的全局注意力机制进行专门优化，仍然是一个处于探索阶段的前沿领域。

目前已存在多个开源的存算一体（CIM）仿真平台，包括针对基于 SRAM/RRAM 架构的 NeuroSim [Peng et al. 2020]、用于忆阻器阵列仿真的 MemTorch，以及由 IBM 开发、用于相变存储器（PCM）硬件感知训练的 AIHWKIT。

本研究在三个方面与这些工作有所不同：
1. 本研究专门针对有机光电器件，并集成了显式的光响应物理建模；
2. 提供了一个基于配置文件（Profile-driven）的器件替换接口，可适配任意新型器件技术；
3. 在统一的实验协议下，跨 CNN 和 Transformer 骨干网络对推理精度、保持力和能耗进行了联合评估。

本研究借鉴了早期 Alibart 等人对有机忆阻器进行的蒙特卡洛（Monte Carlo）仿真思路，但通过 PyTorch 框架将其扩展到了更复杂的模型和更真实的物理场景。我们不仅验证了 HAT 在补齐有机设备物理缺陷中的有效性，还系统地揭示了现代视觉模型在面对状态相关噪声（Proportional Noise）和非线性权重更新（Non-linear Writing）时的稳健性边界。
